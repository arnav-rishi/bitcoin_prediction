Model Architecture Details

Component Specification
Model Type Long Short-Term Memory (LSTM) Neural Network
Input Features Close Price, Volume % Change, SMA Difference, RSI
Sequence Length 60 days
LSTM Layer 1 100 units, return_sequences=True
Dropout Layer 1 Rate = 0.2
LSTM Layer 2 100 units, return_sequences=False
Dropout Layer 2 Rate = 0.2
Output Layer Dense(1) - Linear activation
Optimizer Adam (learning_rate = 0.001)
Loss Function Mean Squared Error (MSE)
Batch Size 64
Early Stopping Patience = 10 epochs

Indicator Description Window Period

SMA_diff Normalized difference between current price and SMA 15 days
